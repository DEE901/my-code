# Minecraft Clone

## Installation Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/DEE901/my-code.git
   ```
2. Navigate to the project directory:
   ```bash
   cd my-code
   ```
3. Install the required dependencies:
   ```bash
   npm install
   ```

## Usage Instructions
1. Start the game:
   ```bash
   npm start
   ```
2. Open your browser and go to `http://localhost:3000` to play the game.

3. Use the following keys for controls:
   - `W` - Move Forward
   - `A` - Move Left
   - `S` - Move Backward
   - `D` - Move Right
   - `Space` - Jump

Enjoy playing!
